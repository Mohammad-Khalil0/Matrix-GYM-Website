## Live Demo

After GitHub Pages is enabled for this repository, the site will be available at:

`https://mohammadkhalil5.github.io/COSC333-Project/`

## Pages Included

- `homepage.html` - main landing page
- `membershipPage.html` - membership and payment form
- `ProductPage.html` - product catalog
- `ProgramPage.html` - training programs and booking modal
- `aboutUsPage.html` - short project about page

## Run Locally

Open `index.html` in a browser, or use a simple local server.

## Publish With GitHub Pages

1. Push the latest `main` branch to GitHub.
2. Open the repository settings on GitHub.
3. Go to `Settings > Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select branch `main` and folder `/(root)`.
6. Save, then wait for GitHub Pages to publish the site.

According to GitHub Docs, GitHub Pages looks for an entry file such as `index.html`, and publishing can be configured from a branch in the repository root.
