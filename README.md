# Matrix Gym

Matrix Gym is a front-end web project built with HTML, CSS, and JavaScript as part of my COSC 333 Web Programming coursework at Rafik Hariri University in Fall 2024.

This project demonstrates my ability to build a complete multi-page website from scratch. It includes a landing page, membership form, product catalog, training programs page, and about page, and it highlights skills I developed in semantic HTML structure, responsive CSS styling, and JavaScript interactivity such as form validation, filtering, modals, and navigation behavior.

## Live Demo

`https://mohammad-khalil0.github.io/Matrix-GYM-Website/`

## Project Pages

- `index.html` - root entry point that redirects to the homepage
- `homepage.html` - main landing page
- `membershipPage.html` - membership and payment form
- `ProductPage.html` - supplements and equipment catalog
- `ProgramPage.html` - training programs, comments, and booking modal
- `aboutUsPage.html` - short about page

## Run Locally

Because this is a static site, you can run it in either of these ways:

1. Open `index.html` directly in a browser.
2. Start a simple local server from the project folder and open the local URL.
